# Skill: UI Validation (optional)

## Purpose
Сделать UI поведение «видимым» для агента: скриншоты, DOM snapshots, basic E2E.

## Default
- Если проект web: используйте Playwright/Cypress.
- Если mobile/desktop: базовые скриншоты и smoke-journeys.

## Output
- артефакты: screenshots / видео (если пайплайн поддерживает)
- список проверенных user journeys
